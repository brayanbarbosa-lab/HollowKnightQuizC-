## 📋 Sobre o Projeto

O **Hollow Knight Quiz** é um jogo de perguntas e respostas sobre o universo de **Hollow Knight**, desenvolvido em **C# com Windows Forms**. Cada pergunta tem 3 alternativas, feedback visual de acerto ou erro e sistema de pontuação acumulada.

> 🎓 Projeto desenvolvido como trabalho escolar no curso de Desenvolvimento de Sistemas — **2DSB ETEC JA, 2024**.

---

## ✨ Funcionalidades

- ❓ **10 perguntas** sobre o universo de Hollow Knight
- 🖼️ **Imagens temáticas** em cada pergunta com arte do jogo
- ✅ **Feedback visual** — imagem de certo ou errado ao responder
- 🔒 **Bloqueio de resposta** — não é possível alterar após confirmar
- 🔢 **Sistema de pontuação** — acumula pontos a cada acerto
- 🎨 **Visual imersivo** — backgrounds e imagens do jogo em cada tela
- 🏆 **Resultado final** — veja sua pontuação ao terminar o quiz

## 🛠️ Como Rodar o Projeto

### Pré-requisitos

- [Visual Studio](https://visualstudio.microsoft.com/) com suporte a **.NET Framework**

### 📦 Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/brayanbarbosa-lab/HollowKnightQuizC-.git

# 2. Abra o arquivo .sln no Visual Studio
```

### ▶️ Rodando

1. Abra o projeto no **Visual Studio**
2. Pressione **F5** ou clique em **Iniciar**
3. Clique em **Iniciar Quiz** e responda as perguntas!
